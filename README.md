### 🚀 About Me

I'm a self-taught builder based in Lahore, Pakistan, currently pursuing a **BS in Data Science** at the University of Central Punjab. My approach to development is a little different — instead of relying on one tool, I **direct multiple AI platforms** (Claude, ChatGPT, Gemini, Grok, Kimi), matching each one to the task it's best at, to design, build, and ship working software end-to-end.

This isn't about prompting for fun — it's a workflow. I use it to architect applications, write and debug code, and go from idea to a finished, packaged product fast. Case in point: I built and packaged a **50+ language PDF translator as a standalone Windows app in 2 days, solo, with zero paid tools**.

I also write about how I build — I authored a published guide on practical AI use, itself an experiment in AI-driven content production.
